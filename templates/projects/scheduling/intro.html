Scheduling Shifts for a Bank

Problem

How can we assign employees to suitable shifts in a financial institution to minimize labor costs while considering employee skill levels and salary?

Parameters and Variables

To simplify the problem and ensuring the scalability of the problem, we have to define the parameters and variables.

Parameter for the shifts:

  1. Shift j: There are seven types of shifts
  2. Weekdays w: We will assume that all bank employees only work five days a week, so the days are enumerated to \( 0 \leq w \leq 4 \)
  3. Demand \( d_{jw} \): The demand for each weekdays are different for each type of shifts.

Parameter for the shifts:

  1. Employee i: Each employee will have a unique id.
  2. Skill levels l: The skill levels will be divided into four levels, \( 0 \leq l \leq 3 \)
  3. Training \( t_{ij} \): This is an array of length 7, which is the number of the types of shift
  4. Salary s: The salary of each employee is related to the training level, defined as \[ s = k + a \sum_{i}(t_{i,1}+t_{i,2}) + b\sum_{i}(t_{i,3}) + c\sum_{i}(t_{i,4}+t_{i,5}+t_{i,6}) + d\sum_{i}(t_{i,7}) \]

Decision variables:

In optimization problems, decision variables are the variables that are changable to determine the goal of the problem. In order to get the optimal result, the solver needs to determine a value for each decision variables.

  • Let \( x_{ijd} \in (0,1) \) be a binary variable, where \( x_{ijd} = 1 \) if employee i is assigned to shift j at day d, and 0 otherwise.

Formulation

Objective

\[ \sum_{i}\sum_{j}\sum_{d}(x_{ijd} * s_{i}) \]

\( s_{i} \) is the salary of employee i

Constraints

  1. An employee can be assigned to at most one shift for each day

    \[ \sum_{i}\sum_{d}(x_{ijd} \leq 1) \forall(j) \]

  2. Each shifts has the required number of employees at day d

    \[ \sum_{i}(x_{ijd}) = d_{jd} \forall(j), \forall(d) \]

  3. Ensure that only employees with the required skill level can be assigned to shifts
  4. For each shift, select the employee with the highest training level
  5. Ensure that the selected employee has the lowest salary among those with the highest training level for each shift
